การหาลิมิตของฟังก์ชันพหุนาม (แทนค่าโดยตรง)

ทฤษฎีบทการแทนค่าตรง (Direct Substitution)

ฟังก์ชันพหุนามเป็นฟังก์ชันที่มีความต่อเนื่องในทุกๆ จุด ดังนั้น ในการหาลิมิตของฟังก์ชันพหุนามเมื่อ x เข้าใกล้ค่าคงตัว c เราสามารถ "แทนค่า x ด้วย c เข้าไปในฟังก์ชันตรงๆ ได้เลย"

limx→c P(x) = P(c)

เลือกรูปแบบฟังก์ชันพหุนามและค่าคงตัวเพื่อดูการหาค่าลิมิต:

ขั้นตอนการคำนวณแทนค่าตรง

limx→1 (x² + 2x - 3)
แทนค่า x = 1:
= (1)² + 2(1) - 3
= 0
3. ลิมิตฝั่งซ้ายและฝั่งขวา
3 / 9

ลิมิตซ้าย-ขวา และการมีอยู่ของลิมิต

ลิมิตจะหาค่าได้เมื่อไร?

ลิมิตของฟังก์ชัน ณ จุดใดๆ จะหาค่าได้ก็ต่อเมื่อ "ลิมิตซ้ายและลิมิตขวาต้องลู่เข้าหาค่าเดียวกัน"

นิยามทางสัญลักษณ์:
• ลิมิตซ้าย: limx→a⁻ f(x) = L₁ (เข้าใกล้ฝั่ง x < a)
• ลิมิตขวา: limx→a⁺ f(x) = L₂ (เข้าใกล้ฝั่ง x > a)
*ลิมิตหาค่าได้ ก็ต่อเมื่อ L₁ = L₂ = L

ตัวอย่าง: ฟังก์ชันขั้นบันได (Step Function)

ลองเลื่อนจุด x เพื่อพิจารณาลิมิตที่จุดรอยต่อ x = 2:

ลิมิตฝั่งซ้าย (x → 2⁻): limx→2⁻ f(x) = 2.00
ลิมิตฝั่งขวา (x → 2⁺): limx→2⁺ f(x) = 4.00
ลิมิตที่จุดรอยต่อ: limx→2 f(x) ไม่มีค่า (เพราะ 2 ≠ 4)

กราฟ Step Function: f(x) = 2 เมื่อ x < 2 และ f(x) = 4 เมื่อ x ≥ 2

ค่าจริง f(x) = 2.00
4. คุณสมบัติสำคัญของลิมิต
4 / 9

ทฤษฎีบทและสมการคุณสมบัติสำคัญของลิมิต

กำหนดให้ limx→c f(x) = L และ limx→c g(x) = M และให้ k เป็นค่าคงตัวใดๆ:

1. ค่าคงตัว (Constant)
limx→c k = k
2. ตัวแปรเอกลักษณ์
limx→c x = c
3. ดึงตัวคูณคงตัว
limx→c [k·f(x)] = k·L
4. การบวก (Sum)
limx→c [f(x) + g(x)] = L + M
5. การลบ (Difference)
limx→c [f(x) - g(x)] = L - M
6. การคูณ (Product)
limx→c [f(x)·g(x)] = L·M
7. การหาร (Quotient)
limx→c f(x)g(x) = LM (เมื่อ M ≠ 0)
8. ยกกำลัง (Power)
limx→c [f(x)]ⁿ = Lⁿ
9. ถอดราก (Root)
limx→c ⁿ√f(x) = ⁿ√L (เมื่อ ⁿ√L หาค่าได้)

วิธีการแก้ปัญหาเบื้องต้น (Suggested Methods for 0/0)

1. การแยกตัวประกอบ (Factoring Method)
  • แยกตัวประกอบพหุนามทั้งเศษและส่วน
  • ตัดทอนตัวประกอบที่ทำให้เกิดศูนย์ออก (เช่น x - a)
  • แทนค่าเป้าหมายเพื่อหาคำตอบของลิมิต
2. การคูณด้วยสังยุค (Conjugate Method)
  • ใช้ในกรณีที่มีพจน์ติดเครื่องหมายราก (Square Root)
  • คูณด้วยสังยุค (เช่น คูณเศษและส่วนด้วย √x + a)
  • จัดรูปกำลังสองสมบูรณ์เพื่อตัดทอนตัวศูนย์แล้วแทนค่า

กราฟแสดง "จุดโหว่" (Hole) ของ f(x) = x² - 4x - 2 ที่ x = 2

f(x) = 3.50 ลิมิตจริง = 4.00

ความต่อเนื่องของฟังก์ชัน (Continuous Functions)

เงื่อนไขความต่อเนื่อง 3 ข้อที่จุด x = a

ฟังก์ชัน f(x) จะมีความต่อเนื่องที่จุด x = a ก็ต่อเมื่อสอดคล้องกับเงื่อนไขครบทั้ง 3 ข้อดังนี้:

✓ 1. f(a) หาค่าได้ (มีจุดระบายทึบที่พิกัด x = a)
✓ 2. limx→a f(x) หาค่าได้ (ลิมิตซ้ายและขวาเท่ากัน)
✓ 3. limx→a f(x) = f(a) (ค่าลิมิตเท่ากับค่าจริงที่จุดนั้นพอดี)

เลือกประเภทกราฟเพื่อตรวจสอบความต่อเนื่อง:

กราฟฟังก์ชันที่จุด x = 2

ฟังก์ชันต่อเนื่องที่ x = 2 (กราฟไม่ขาดตอน)
7. ความชันและบทนำสู่การหาอนุพันธ์

จากอัตราการเปลี่ยนแปลงเฉลี่ย สู่อนุพันธ์ของฟังก์ชัน

แนวคิดของเส้นสัมผัสเส้นโค้ง

สมมติฟังก์ชัน f(x) = 0.25x² + 1 เราหาความชันเฉลี่ยของเส้นผ่านจุด P(2, 2) และ Q(2+h, f(2+h)) ด้วยสูตร:

ความชัน Secant = f(x+h) - f(x) h

เมื่อเราลดขนาดของ h ลงเรื่อยๆ จนเกือบเป็นศูนย์ (h → 0) เส้นตัด (Secant Line) จะหมุนมาเป็น เส้นสัมผัสโค้ง (Tangent Line) ณ จุด P พอดี

ค่าความชันสัมผัส (อนุพันธ์ที่ x=2):
• ความชันเฉลี่ย Secant = 1.50
• ความชันสัมผัส Tangent (เมื่อ h → 0) = 1.00
ลิมิตความชันนี้เรียกว่า "อนุพันธ์" (Derivative):
f'(x) = limh→0 f(x+h) - f(x) h

สาธิตการเปลี่ยนเส้นตัดเป้นเส้นสัมผัส (เมื่อ h → 0)

จุด P = (2, 2) | จุด Q = (4.00, 5.00)
8. คุณสมบัติและสูตรการหาอนุพันธ์

สูตรและคุณสมบัติพื้นฐานของการหาอนุพันธ์ (ดิฟ)

ให้ c และ n เป็นค่าคงตัว และให้ u, v เป็นฟังก์ชันของตัวแปร x:

1. ดิฟค่าคงที่ (Constant Rule)
ddx(c) = 0
2. ดิฟเทียบตัวแปรเอง (Identity Rule)
ddx(x) = 1
3. กฎกำลัง (Power Rule) - สำคัญมาก!
ddx(xⁿ) = n·xⁿ⁻¹
4. ดึงสัมประสิทธิ์คงตัว
ddx[c·u] = c · dudx
5. ดิฟแยกพจน์ บวก/ลบ (Sum & Difference)
ddx[u ± v] = dudx ± dvdx

ตัวอย่างการดิฟพจน์เดี่ยว:

• ddx(x²) = 2x
• ddx(x³) = 3x²
• ddx(5x⁴) = 5·(4x³) = 20x³

ตัวอย่างการดิฟพหุนามหลายพจน์:

ddx(4x³ + 5x² + 4)
= ddx(4x³) + ddx(5x²) + ddx(4)
= 12x² + 10x + 0 = 12x² + 10x
