⚙️ Customize Lessons (Teacher Panel)

▼

This portal is fully config-driven, making it incredibly simple to add new modules, quizzes, or simulators. To customize the lesson list, open the file lessons.js in the project directory.

You can append a new object to the LESSONS array with the following format:

{
  id: "lesson-name",
  title: "Lesson Title",
  description: "A short summary...",
  category: "Theory / Practical / Quiz",
  difficulty: "Beginner / Intermediate / Advanced",
  time: "10 mins",
  url: "path_to_file.html",
  status: "available", // "available" or "coming-soon"
  icon: "📚", // Emoji or SVG
  accentColor: "cyan" // "cyan", "purple", "amber", "red"
}
