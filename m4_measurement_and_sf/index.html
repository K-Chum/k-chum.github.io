เลขนัยสำคัญ (Significant Figures) คืออะไร?

ความหมาย

เลขนัยสำคัญ คือ ตัวเลขที่ได้จากการวัดจริง ๆ ในกระบวนการทางวิทยาศาสตร์ ซึ่งประกอบด้วย ตัวเลขที่อ่านได้ตรงจากเครื่องมือ และ ตัวเลขที่ประมาณขึ้นมาด้วยสายตาอีก 1 ตัว เท่านั้น

เลขนัยสำคัญเป็นตัวบ่งบอก "ความละเอียดของเครื่องมือวัด" ยิ่งเลขนัยสำคัญมีจำนวนหลักมาก แสดงว่าใช้เครื่องมือวัดที่มีความละเอียดสูงขึ้น

*จำไว้เสมอ: ตัวเลขทุกตัวจากการวัดมีความหมายในทางฟิสิกส์ ไม่ใช่แค่ทศนิยมตกแต่ง!

หลักการนับเลขนัยสำคัญ (Counting Rules)

สรุปกฎการนับเลขนัยสำคัญ

  1. เลขทุกตัวที่ไม่ใช่ 0 ให้นับหมด
    • ตัวอย่าง: 458 → 3 ตัว, 1.234 → 4 ตัว
  2. กรณีของเลข 0 มีกฎ 3 ข้อดังนี้:
    • 2.1 เลข 0 ด้านหน้า (ซ้ายสุด) ไม่นับ! ทำหน้าที่แค่ทศนิยมบอกขนาดหลัก
      ตัวอย่าง: 0.0035 → 2 ตัว
    • 2.2 เลข 0 ตรงกลาง (ระหว่างตัวอื่น) ให้นับเสมอ!
      ตัวอย่าง: 405 → 3 ตัว, 1.009 → 4 ตัว
    • 2.3 เลข 0 ด้านหลัง (ขวาสุด):
      • จำนวนเต็ม: นับหรือไม่นับก็ได้ (คลุมเครือ) ควรหลีกเลี่ยงโดยเขียนในรูป สัญกรณ์วิทยาศาสตร์
        ตัวอย่าง: 4500 อาจเป็น 2, 3 หรือ 4 ตัว
        (4.5 × 103 → 2 ตัว, 4.50 × 103 → 3 ตัว, 4.500 × 103 → 4 ตัว)
      • ทศนิยม: ให้นับเสมอ เพราะแสดงถึงความละเอียดตั้งใจวัด
        ตัวอย่าง: 2.30 → 3 ตัว, 0.0500 → 3 ตัว

กฎสำคัญ: ยึด "ตำแหน่งทศนิยม" ที่น้อยที่สุด

ผลลัพธ์จากการบวกและลบตัวเลขที่ได้จากการวัด จะต้องมี จำนวนตำแหน่งทศนิยมเท่ากับตัวตั้งที่มีจำนวนทศนิยมน้อยที่สุด

หลักการคูณและหารเลขนัยสำคัญ

กฎสำคัญ: ยึด "จำนวนเลขนัยสำคัญ" ที่น้อยที่สุด

ผลลัพธ์จากการคูณและหาร จะต้องมี จำนวนเลขนัยสำคัญทั้งหมด เท่ากับตัวตั้งที่มีจำนวนเลขนัยสำคัญรวมน้อยที่สุด

ความแตกต่างสำคัญกับการบวก/ลบ:
การคูณ/หาร ไม่ได้ยึดตำแหน่งทศนิยม แต่ยึด จำนวนหลัก (นัยสำคัญรวม) ของตัวคูณที่น้อยที่สุด
วิธีคำนวณ:
  1. คูณหรือหารตัวเลขตามปกติ
  2. นับจำนวนเลขนัยสำคัญของตัวตั้งแต่ละตัว
  3. ปัดเศษผลลัพธ์ให้มีหลักเลขนัยสำคัญเท่ากับจำนวนที่น้อยที่สุดนั้น

ความคลาดเคลื่อนของการวัด (Uncertainty)

ผลจากการวัดในความเป็นจริง

ในการทดลองทาง ไม่มีข้อมูลวัดใดที่ถูกต้องสมบูรณ์แบบ 100% ทุกค่าที่ได้จากการทดลองจะมีความคลาดเคลื่อนติดมาเสมอด้วยปัจจัยของเครื่องมือ สภาพแวดล้อม หรือขีดจำกัดของผู้ทดลอง

ดังนั้น เราจึงต้องรายงานผลที่ได้จากการวัดโดยรวมเอาความคลาดเคลื่อนที่เกิดขึ้นในรูปของ:

A ± ΔA

โดยที่ A คือ ค่ากลางที่วัดได้ และ ΔA คือ ค่าความคลาดเคลื่อน

ช่วงค่าคำตอบที่เป็นไปได้

การรายงานผลเป็น A ± ΔA หมายความว่า ค่าจริง ๆ จะอยู่ในช่วง [A - ΔA, A + ΔA] ทดลองปรับค่าด้านล่างเพื่อสังเกตช่วงความคลาดเคลื่อนบนสเกลเครื่องมือวัด:

การรายงานผล: 25.0 ± 0.2 °C
ช่วงที่เป็นไปได้: 24.8 °C ถึง 25.2 °C
แถบสีส้มและเส้นปะแสดงช่วงความคลาดเคลื่อนที่เป็นไปได้ทั้งหมดของอุณหภูมิจริง

การบวกและลบตัวเลขที่มีค่าคลาดเคลื่อน

หลักการ: ค่าคลาดเคลื่อนสะสมเพิ่มขึ้นเสมอ

เมื่อนำปริมาณสองตัวมาบวกหรือลบกัน ค่าคลาดเคลื่อนของคำตอบจะได้จากการนำค่าคลาดเคลื่อนของตัวตั้งทุกตัวมารวมกันเสมอ (ไม่มีการนำมาลบกันเด็ดขาด!)

สูตร:
ตัวตั้งบวกกัน: (A ± ΔA) + (B ± ΔB) = (A + B) ± (ΔA + ΔB)
ตัวตั้งลบกัน: (A ± ΔA) - (B ± ΔB) = (A - B) ± (ΔA + ΔB)
ตัวอย่าง:
  • บวก: (5.0 ± 0.2) + (3.0 ± 0.1) = (5.0 + 3.0) ± (0.2 + 0.1) = 8.0 ± 0.3
  • ลบ: (5.0 ± 0.2) - (3.0 ± 0.1) = (5.0 - 3.0) ± (0.2 + 0.1) = 2.0 ± 0.3
*ข้อระวัง: แม้จะลบค่าจริงกัน แต่ค่าคลาดเคลื่อนยังคงต้องนำมาบวกกัน!

การคูณและหารตัวเลขที่มีค่าคลาดเคลื่อน

หลักการ: นำเปอร์เซ็นต์ความคลาดเคลื่อนมารวมกัน

เมื่อนำตัวเลขที่มีความคลาดเคลื่อนมาคูณหรือหารกัน เปอร์เซ็นต์ค่าคลาดเคลื่อนของคำตอบสุดท้าย จะมีค่าเท่ากับผลรวมของเปอร์เซ็นต์ค่าคลาดเคลื่อนของตัวประกอบแต่ละตัว

ขั้นตอนคำนวณ:
  1. คำนวณหาเปอร์เซ็นต์คลาดเคลื่อนของแต่ละตัว:
    %ΔA = (ΔA / A) × 100%
  2. นำเปอร์เซ็นต์ความคลาดเคลื่อนมาบวกกันเสมอ:
    %ΔY = %ΔA + %ΔB
  3. เขียนผลลัพธ์ในรูป: ค่าจริง (คลาดเคลื่อน ± %)
ตัวอย่าง:
  • ความกว้าง A = 5.0 ± 0.1 m → %ΔA = (0.1 / 5.0) × 100% = 2%
  • ความยาว B = 10.0 ± 0.2 m → %ΔB = (0.2 / 10.0) × 100% = 2%
  • ผลลัพธ์ A × B หรือ A ÷ B จะมีเปอร์เซ็นต์ความคลาดเคลื่อนรวม: 2% + 2% = 4%
  • คำตอบ: 50 (คลาดเคลื่อน ± 4%)
